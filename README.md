## Scripts do projeto

### Instalação

```bash
$ npm install
```

### Desenvolvimento

```bash
$ npm run dev
```

### Build

```bash
# Para windows
$ npm run build:win

# Para macOS
$ npm run build:mac

# Para Linux
$ npm run build:linux
```
